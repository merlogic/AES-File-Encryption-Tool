# SecureVault - AES File Encryption Tool

A Python-based desktop application that allows users to securely encrypt and decrypt files using AES encryption with a modern GUI built using Tkinter.

>  This project is currently under development. 
    More features will be added soon.

---

##  Features

-  Select any file from system
-  Encrypt files using AES encryption
-  Decrypt encrypted files back to original
-  Secure key handling using environment variables
-  Beginner-friendly encryption project
